## Installation (5 min)

1. Télécharge **TestFlight** sur ton iPad : [apps.apple.com/app/testflight/id899247664](https://apps.apple.com/app/testflight/id899247664)
2. Ouvre ce lien d'invitation : **[LIEN_TESTFLIGHT]**
3. Accepte l'invitation → installe Playco

## Premier usage (15 min)

1. Ouvre l'app → clique **« Créer mon équipe »**
2. Complète le wizard en 6 étapes (établissement, sport, profil coach, équipe, joueurs, calendrier)
3. Note bien les identifiants auto-générés affichés dans la sheet récap finale (tu peux aussi les retrouver dans **Paramètres → Identifiants de l'équipe**)
4. À la sortie du wizard, choisis ton plan :
   - **Playco Pro** (14,99 $/mois) — coach + staff
   - **Playco Club** (25 $/mois) — Pro + accès app athlètes
   - **Essai 14 jours** offert en TestFlight, **aucune carte requise**

> ⚠️ En TestFlight, les durées Apple sont accélérées : 14 jours = ~3 min, 1 mois = ~5 min, 1 an = ~1 h. Pratique pour tester les cycles essai → expiration → renouvellement rapidement.

## Tests demandés (2-3 semaines)

- [ ] Ajoute tes **vrais matchs** et saisis les stats live pendant une partie
- [ ] Teste la **composition match** (6 de départ + rotation)
- [ ] Partage l'identifiant d'un athlète → demande-lui de se connecter à l'app (plan Club uniquement)
- [ ] Exporte une **feuille de match PDF** après un match
- [ ] Exporte tes **stats en CSV** (Équipe → Exporter → Joueurs)
- [ ] Joue avec la **heatmap** et les **analytics saison**
- [ ] Essaie l'app **hors ligne** pendant un déplacement
