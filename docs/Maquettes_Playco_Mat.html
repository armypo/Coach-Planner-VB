01 — Manifeste

Playco n'est pas une app de sport. C'est l'instrument de travail d'un coach. Chaque écran se lit comme un document imprimé avec soin : de l'encre sur du papier, des chiffres qui tombent juste, des traits qui veulent dire quelque chose. Rien ne brille, rien ne clignote, rien ne décore. La seule couleur qui compte est celle de l'équipe ; le seul rouge est celui du direct. Le calme de l'interface, c'est le respect du travail du coach.

02 — Couleur & matière

L'encre, le papier, une seule couleur

Le verre est délégué au chrome système ; tout le contenu est opaque. La profondeur vient du papier (trois niveaux), pas de l'ombre. Aucune information n'est portée par la couleur seule.

07 — Les dix lois

Vérifiables en revue de code

1L'image fonctionnelle. Toute image appartient aux listes fermées (16 glyphes, 8 outils, 5 onglets). Le reste disparaît.
2L'accent unique. La seule couleur du contenu est celle de l'équipe. Exception : les postes, sur les jetons du terrain seulement.
3Le chiffre tabulaire. Tout nombre comparable est tabulaire, aligné à droite.
4Le verre-chrome. Le verre n'existe que sur la tab bar, les toolbars et les présentations système.
5Le papier. Fonds unis par tokens ; toute surface porte son filet de 0,5 pt.
6Le style nommé. Onze tokens typographiques, aucune taille libre, jamais d'arrondi.
7L'ombre unique. Seul le flottant projette une ombre.
8Le calme. Rien ne bouge sans cause utilisateur ; les chiffres roulent, les cadres jamais.
9La redondance. Aucune information portée par la couleur seule — plein/contour, V/D, signes, « DIRECT ».
10Le gymnase. Courtside : noir pur, AAA, cibles 60 pt, boutons en mots.
